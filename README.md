## 说明
### menu
1. tit  str 标题  
2. type str 标题唯一id
3. open bool 是否默认展开
4. hide bool 是否隐藏
4. pwd  bool 是否设置密码
5. memory bool 是否记忆
6. list arr 文件列表
### html
1. index 首页
2. content 内容页面
3. fxmc		佛学名词页面
4. maokan 	猫砍目录页面
5. maokan_content 猫砍内容页面(未做)
